# 《凯撒：共和国最后的春天》完整故事局剧本 v1.0（持续叙事流版）

> 英文名：**CAESAR: THE LAST SPRING OF THE REPUBLIC**  
> 文档类型：故事局剧本 / 可配置历史推演脚本  
> 适用产品：Many Worlds / AI 多人命运推演 / 剧情消息驱动的动态决策 Web  
> 首版玩法：Web MVP，单人模式优先  
> MVP 主控角色：马库斯·尤尼乌斯·布鲁图斯（Marcus Junius Brutus）  
> 支持扩展：3—6 人异步角色局，缺席角色由 AI 接管  
> 游戏结构：7 幕制；第 1—6 幕各 3—5 条核心剧情消息、固定 2 次关键决策、最多 2 次主动谋划；第 7 幕进入“罗马裁决”；整局共 12 次关键决策  
> 题材类型：古罗马历史权谋 / 共和国危机 / 友情与背叛 / 舆论与军权 / 架空历史  
> 预计时长：单人 40—60 分钟；多人异步 1—3 天  
> 历史范围：公元前 44 年卢佩卡利亚节至凯撒葬礼前后  
> 重要说明：本剧本以真实历史人物和事件为锚点，吸收普鲁塔克、阿庇安、苏维托尼乌斯与莎士比亚《尤利乌斯·凯撒》的经典叙事结构；为满足多人推演与分支玩法，对时间、会面、情报流转和人物参与度进行了压缩与架空。它不是史实复现，也不是任何现代影视作品的改编。

# 0A. v1.0 产品结构说明

本剧本继承《桑田诏：嘉靖财政危局》的核心产品结构：

```text
连续故事
→ A/B/C 三个标题式建议
→ 玩家输入自己的完整决定
→ AI 正在推演局势……
→ 结果故事
→ 本次变化
→ 下一段故事
```

玩家可见选项只显示标题。

以下内容只允许存在于机器配置、导演规则和测试用例中，不得在前台选项中展示：

```text
选项完整说明
可能收益
可能风险
成功率
数值补丁
角色真实反应
结局倾向
隐藏暗线
```

结果发生后，玩家可以看到：

```text
结果故事
公开的关系变化
公开的世界变化
已经暴露的风险
由自己造成的他人回响
```

不能提前展示“这个选择会增加多少成功率”或“这是最佳选项”。

# 0B. 固定决策与主动谋划

机器配置统一采用：

```text
第 1—6 幕各 2 个关键决策
第 1—6 幕每天最多 2 次主动谋划
第 7 幕无普通决策、无主动谋划
整局固定 12 个关键决策
主动谋划理论上限 12 次
```

统一前台术语：

```text
关键决策
主动谋划
谋划机会
自拟谋划
执行谋划
```

禁止使用：

```text
AP
行动点
体力值
筹谋点
```

# 0. 剧本使用方式

## 0.1 这不是传统剧本杀

本剧本不是：

```text
找出固定凶手
按照角色台词演完
好人阵营对抗坏人阵营
投票淘汰某个玩家
复现凯撒必然被刺杀的历史结果
```

它是一套可被 AI 因果引擎读取的“历史命运局”。

玩家可以：

```text
继续历史上的密谋
试图阻止密谋
逼迫凯撒接受权力限制
与安东尼、卡西乌斯或西塞罗秘密交易
改变凯撒进入元老院的条件
在凯撒倒下后争夺罗马的解释权
让凯撒活下来，却令共和国名存实亡
让凯撒死去，却避免历史上的舆论反噬
```

历史只提供初始局势，不保证最终结果。

---

## 0.2 玩家看到的页面

```text
左侧：
我的身份
公开目标
私人目标
资源
筹码
关系
当前风险

中间：
持续叙事时间线
关键决策
AI 推演结果
本次变化
重要事件
罗马回声

右侧：
今日谋划
可接触人物
正在推进
可调查方向
可用筹码
自拟谋划
```

中间区域不显示系统日志：

```text
卡西乌斯选择了【扩大密谋】
安东尼执行了【控制广场】
```

而是转译成角色视角故事：

```text
夜里，一名你并不认识的元老从卡西乌斯宅邸侧门离开。

他没有带随从，也没有走常用的街道。

不久以后，凯撒家中的一名书记员收到消息：
最近询问元老院席位安排的人，比往常多了三个。

你无法确定卡西乌斯在扩大联盟，还是有人正在故意让密谋暴露。
```

## 0.3 玩家输入方式

每个关键决策展示：

```text
你要如何应对？

A 行动标题
B 行动标题
C 行动标题

你也可以写下自己的决定：

[输入你的处理方式……]
0 / 200

提交决策
```

规则：

- A/B/C 仅显示行动标题；
- 点击建议后，标题自动写入输入框；
- 玩家可以继续补充，也可以完全重写；
- 玩家可以直接输入自定义行动；
- 最终文本最多 200 字；
- 自定义决策必须符合身份、权力、资源、地点和时代条件；
- `CUSTOM` 仅作为后台 `optionKey`，前台不显示“自定义选项 D”。

## 1.2 一句话钩子

中文：

> 凯撒被授予终身独裁权，罗马开始害怕他会成为国王。你是布鲁图斯：凯撒信任你，密谋者需要你，共和国却没有告诉你该救谁。

英文：

> **Caesar trusts you. The conspirators need you. Rome expects you to choose which one must survive.**

---

## 1.3 MVP 玩家身份

单人模式默认控制：

```text
马库斯·尤尼乌斯·布鲁图斯
```

你不是普通元老。

你同时拥有：

```text
古老共和家族的声望
凯撒给予的赦免与信任
元老院中的职位
卡西乌斯的私人关系
罗马公众对“布鲁图斯之名”的期待
接近凯撒核心圈的资格
```

你也同时被四种力量拉扯：

```text
凯撒希望你证明忠诚
卡西乌斯希望你赋予密谋正当性
西塞罗希望制度重新高于个人
安东尼怀疑你正在动摇
罗马公众把你当成共和传统的象征
```

你的核心问题不是：

> “是否参加刺杀？”

而是：

> **当法律已经无法限制一个最有能力的人时，你能否阻止他，而不把罗马交给更危险的人？**

---

## 1.4 剧本核心冲突

公开冲突：

```text
凯撒获得终身独裁权。
越来越多元老相信，他正在把共和国变成个人统治。
```

真实冲突：

```text
凯撒一旦失去权力，谁能控制元老院、军团、城市秩序和群众？
```

底层冲突：

```text
所有人都说自己在拯救共和国，
但每个人想拯救的共和国都不一样。
```

角色对“共和国”的不同理解：

```text
凯撒：
结束无休止的内战、债务与派系斗争，比旧制度形式更重要。

布鲁图斯：
个人权力必须受到限制，但秩序不能靠另一场内战恢复。

卡西乌斯：
只要凯撒仍掌握军队和任命权，任何承诺都不可信。

安东尼：
凯撒的胜利者联盟、老兵和改革不能被旧贵族夺走。

德西穆斯：
共和国、凯撒和密谋都只是局势；真正重要的是谁能在变局中保住兵权与前途。

西塞罗：
没有公开合法性，凯撒和密谋者都只是用力量代替法律。
```

## 1.5 完整故事主线

```text
凯撒胜利归来
→ 安东尼在节庆中递上王冠
→ 凯撒拒绝，但“称王恐惧”扩散
→ 凯撒获得终身独裁权并准备远征
→ 卡西乌斯开始组织反凯撒联盟
→ 伪装成民意的匿名信送到布鲁图斯手中
→ 布鲁图斯决定是否加入、限制或反用密谋
→ 密谋者争论安东尼、西塞罗和行动后的政治安排
→ 凯撒收到多重警告
→ 德西穆斯决定是否把凯撒带到元老院
→ 三月十五日出现刺杀、妥协、揭发或反制分支
→ 各方争夺元老院、军团、广场、遗嘱和公共叙事
→ 罗马决定谁是解放者、叛徒、暴君或新的主人
```

---

## 1.6 剧本体验目标

玩家应该持续感到：

```text
我认识这些人物，但这一次历史不一定照旧发生。
我不是在做“忠诚或背叛”的二选一。
我每一次公开表态，都可能被另一个人用作政治证据。
我可以不接受卡西乌斯和凯撒给出的两条路。
一个正确的道德选择，可能带来灾难性的政治结果。
一个看似卑劣的交易，也可能暂时阻止内战。
凯撒是否活着不是唯一结局；谁能定义罗马才是。
```

## 1.7 历史锚点与架空边界

必须保留的历史锚点：

```text
卢佩卡利亚节上的王冠试探
凯撒获得终身独裁权
凯撒即将离开罗马远征
卡西乌斯推动密谋
布鲁图斯的加入决定密谋合法性
匿名信与“民意”压力
是否同时对付安东尼的争论
卡尔普尔尼娅与多重警告
德西穆斯劝说凯撒出门
三月十五日的元老院会议
凯撒死后关于大赦与法令存续的交易
遗嘱、葬礼和群众态度改变政治结果
```

允许架空：

```text
布鲁图斯提前识破匿名信来源
凯撒向布鲁图斯提出限制权力的秘密协议
安东尼与卡西乌斯进行非正式接触
西塞罗更早参与制度方案
德西穆斯改变立场
凯撒留在家中或主动召开另一场会议
密谋被公开但凯撒选择赦免
刺杀失败或被政治妥协替代
布鲁图斯在事件后成功控制公共叙事
```

禁止无因果跳跃：

```text
玩家一句话让全罗马立刻服从
布鲁图斯无资源直接调动全部军团
卡西乌斯突然无条件忠于凯撒
安东尼没有利益交换就放弃凯撒阵营
凯撒因一次批评主动永久退出政治
现代民主、媒体或司法概念直接照搬进古罗马
角色知道自己历史上的最终结局
```

# 2. 游戏规则

## 2.1 七幕结构

```text
第 1 幕：王冠
第 2 幕：终身独裁者
第 3 幕：写给布鲁图斯的信
第 4 幕：谁必须一起倒下
第 5 幕：三月十五日前夜
第 6 幕：元老院之日
第 7 幕：罗马裁决
```

每幕包含：

```text
1. 开局剧情
2. 3—5 条核心剧情消息
3. 固定 2 次关键决策（第 1—6 幕）
4. 最多 2 次主动谋划（第 1—6 幕）
5. 决策结果与本次变化
6. 角色回响或罗马回声
```

---

## 2.2 核心剧情消息限制

每幕建议：

```text
核心剧情消息：3—5 条
关键决策：2 次
主动谋划：最多 2 次
幕终回响：1 次
```

每条消息至少完成一项功能：

```text
增加时间压力
提供可交易信息
暴露一段关系
改变某人对玩家的判断
产生公开与私密信息差
迫使玩家表态
影响三月十五日或第 7 幕裁决
```

无意义的宴会闲聊、历史百科介绍和重复争论不进入主时间线。

---

## 2.3 AI 角色行动规则

AI 角色不是随机抢戏。

只有在以下条件下行动：

```text
1. 玩家直接影响其利益或安全
2. 历史阶段推动其必须表态
3. 某项关系或风险达到阈值
4. 它掌握的筹码被触发
5. 它发现玩家公开说法与私下行动矛盾
6. 另一角色向它提供了足够交换条件
```

示例：

```text
密谋暴露 ≥ 60
→ 凯撒加强随从、改变会议地点或要求忠诚声明

卡西乌斯信任 ≤ 30
→ 卡西乌斯绕开布鲁图斯扩大密谋

安东尼警觉 ≥ 70
→ 安东尼联络老兵、控制广场并保护凯撒文书

公众对凯撒支持 ≤ 35
→ 凯撒更依赖军团与赦免政策证明合法性

布鲁图斯公共声望 ≥ 75
→ 无论支持哪一方，都能显著改变元老院态度
```

---

## 2.4 每次行动的三重回响

每次关键决策或成功谋划，最多生成：

```text
个人回响：
行动如何改变玩家自己的处境。

他人回响：
至少一名角色因玩家行动获得、失去或误判了什么。

世界回响：
元老院、群众、军团、城市秩序或共和国合法性如何变化。
```

前台示例：

```text
本次变化

个人回响：
凯撒仍然信任你，但开始要求你公开表态。

他人回响：
卡西乌斯认为你在拖延，正在接触更多元老。

世界回响：
“凯撒愿意接受限制”的传闻开始在元老院流动。
```

## 2.5 主动谋划

### 固定规则

```text
第 1—6 幕每幕 2 次谋划机会
成功执行消耗 1 次
被 ActionGuard 拒绝不消耗
未使用不结转
主动谋划不能替代关键决策
完成两次关键决策后才能进入下一幕
```

### 四类谋划

```text
人物交谈
派遣调查
使用筹码
自拟谋划
```

### 主动谋划结果

```text
1 段谋划结果故事
1 组关系 / 线索 / 风险 / 任务 / 筹码变化
1 个 FateSeed 或可追踪痕迹
最多 1 条必要角色反应
```

---

| 幕 | 可接触人物 | 可调查方向 | 可用筹码 / 可能获得 |
|---:|---|---|---|
| 1 | 凯撒、卡西乌斯、西塞罗 | 王冠仪式安排、群众反应、元老缺席名单 | 布鲁图斯家名、凯撒私人信任 |
| 2 | 凯撒、卡西乌斯、安东尼 | 终身独裁授予过程、远征前任命、老兵动向 | 元老院发言权、职位承诺 |
| 3 | 卡西乌斯、波西亚、塞维利娅、西塞罗 | 匿名信来源、密谋参与者、凯撒情报网 | 匿名信、家族声望、私人关系 |
| 4 | 卡西乌斯、德西穆斯、安东尼、西塞罗 | 行动名单、安东尼兵力、行动后制度方案 | 密谋名册片段、凯撒赦免旧恩 |
| 5 | 凯撒、卡尔普尔尼娅、德西穆斯、阿特米多鲁斯 | 警告信、祭兆、会议地点、谁已动摇 | 警告名单、最后通行资格 |
| 6 | 安东尼、西塞罗、元老代表、老兵代表 | 国库钥匙、凯撒文书、广场动向、遗嘱位置 | 元老院合法性、密谋证据、凯撒文书 |

---

## 2.7 主线决策与主动谋划边界

| 对比项 | 关键决策 | 主动谋划 |
|---|---|---|
| 触发方式 | 历史压力与剧情事件触发 | 玩家主动发起 |
| 是否必须 | 必须 | 可选 |
| 每幕数量 | 2 次 | 最多 2 次 |
| 主要影响 | 主线分支与核心因果 | 线索、关系、筹码、伏笔 |
| 是否推进幕次 | 两次完成后可推进 | 不能替代主线 |
| 是否可以改变历史路径 | 可以 | 可以积累条件，但不能一句话改写 |
| 结果位置 | 中间持续叙事时间线 | 中间持续叙事时间线 |

---

## 2.8 多人异步规则

多人版支持 3—6 名玩家：

```text
3 人：
布鲁图斯、凯撒、卡西乌斯
其他角色由 AI 托管

4 人：
增加安东尼

5 人：
增加德西穆斯

6 人：
增加西塞罗
```

每个主轮次：

```text
1. 所有在线玩家收到各自视角故事
2. 玩家提交公开行动或私密行动
3. 超时角色由 AI 按其当前目标接管
4. 系统先进行权限校验与冲突判定
5. 再生成每名玩家的个人、他人、世界回响
6. 只把应公开的事实写入公共时间线
```

同一事件不允许所有玩家看到完全相同的信息。

# 3. 世界变量

## 3.1 核心世界变量

```text
也可以把它理解成任何力量都无法永久替代合法性。

明日，罗马将判断他的克制是真实选择，还是下一次清洗前的停顿。
```

# 第 7 幕：罗马裁决

## 本幕性质

第 7 幕不再提供普通关键决策，也没有主动谋划。

系统根据前六幕所有状态，生成：

```text
1. 事件后的元老院裁决
2. 凯撒法令、遗嘱或协议的处理
3. 广场与群众的态度
4. 军团与老兵的立场
5. 全局结局
6. 布鲁图斯个人下场
7. 其他核心角色下场
8. 可见因果回顾
9. 未解命运种子
```

## 第 7 幕开局：分支文本

### 路径 A：凯撒死亡

```text
【第 7 幕 · 数日后 · 罗马】

凯撒已经不能再进入元老院。

但他的名字出现在每一项争论里。

密谋者要求大赦，并宣称终身独裁已经结束。

安东尼要求凯撒法令继续有效，要求公开遗嘱，也要求一场足以匹配凯撒身份的葬礼。

元老们发现自己陷入一个无法轻易拆开的矛盾：

如果凯撒是暴君，他的法令为何继续有效？

如果他的法令有效，终止他的人凭什么被称为解放者？

如果行动者被赦免，罗马是否承认私人密谋可以替共和国作出最终裁决？

西塞罗看着两边提交的文本，只说：

“今日决定的不是凯撒是否已经死去，而是谁有权解释他的死亡。”
```

### 路径 B：凯撒存活并挫败密谋

```text
【第 7 幕 · 数日后 · 元老院】

参与者的名字已经无法全部隐藏。

凯撒仍掌握军队、法令与行政体系。

安东尼要求彻底清除威胁。

西塞罗却提出：

如果凯撒利用密谋永久取消元老院的制衡，那么密谋者关于个人统治的恐惧将被凯撒自己证明。

罗马等待凯撒选择：

赦免并接受限制；
公开审判；
分化参与者；
或者把这次危机变成共和国最后一次反对他的机会。
```

### 路径 C：宪政妥协

```text
【第 7 幕 · 协议后的第一次会议】

凯撒仍坐在元老院。

但他的席位第一次被一份有期限、有复核和有见证人的协议包围。

支持者称这是凯撒最后一次拯救共和国。

反对者称这只是他为了安全做出的暂时退让。

卡西乌斯没有交出全部名单。

安东尼没有解散老兵渠道。

西塞罗要求把所有条款写入公开法令。

这一次，罗马没有用一个人的倒下来解决矛盾。

它必须证明，纸上的限制是否真的能限制拥有军团的人。
```

### 路径 D：密谋暴露但无人完全胜利

```text
【第 7 幕 · 城市戒备】

行动没有按任何一方计划完成。

部分密谋者离城。

凯撒没有立即全面清算。

安东尼让老兵进入城市周边。

元老院紧急开会，却没有一方拥有足够票数定义事件。

罗马处于一种危险的暂停：

所有人都知道旧秩序已经无法继续，

但新的秩序还没有名字。
```

# 6. 结局系统

## 6.1 全局结局 A：有限权力的共和国

触发参考：

```text
凯撒存活
临时共和国协议完成度 = 3/3
西塞罗认可 ≥ 65
元老院合法性 ≥ 55
内战风险 ≤ 60
凯撒接受至少两项实质限制
安东尼未成功破坏协议
```

结局文本：

```text
【全局结局：有限权力的共和国】

凯撒保留东方远征统帅权，却公开放弃终身独裁的永久表述。

关键任命必须由元老院在限定期限内确认。

普通法令继续有效，老兵土地与平民利益得到保障。

布鲁图斯、凯撒、西塞罗与执政官共同为协议担保。

没有人完全满意。

卡西乌斯认为凯撒仍然掌握太多力量。

安东尼认为元老院正在借恐惧夺回已经失去的权力。

但罗马第一次没有依靠彻底胜利解决危机。

共和国活了下来。

至少活过了这个春天。
```

结局含义：

```text
这是最接近“大胜”的制度结局。
它不是永久和平，而是让未来冲突重新回到可公开争论的制度中。
```

## 6.2 全局结局 B：凯撒幸存，共和国成为外壳

触发参考：

```text
凯撒存活
密谋暴露 ≥ 70
凯撒权威 ≥ 75
军团忠诚 ≥ 75
元老院合法性 ≤ 45
玩家未建立有效协议
凯撒选择有限或全面清洗
```

结局文本：

```text
【全局结局：凯撒幸存，共和国成为外壳】

凯撒公开宣布不会追究所有元老。

他只处理“少数破坏公共和平的人”。

这份克制赢得群众与老兵支持，也让更多中立者离开反对阵营。

元老院继续开会。

法务官继续审理案件。

执政官继续宣布法令。

但从此以后，没有人再相信这些职位能够真正限制凯撒。

共和国的名称被保留下来。

它的最后一次实际反抗，却成为凯撒证明自己必须拥有更多权力的理由。
```

## 6.3 全局结局 C：解放者失去罗马

触发参考：

```text
凯撒死亡
密谋者未控制广场与文书
安东尼保持行动自由
凯撒民望 ≥ 60
凯撒法令与遗嘱公开
布鲁图斯公开解释力不足
```

结局文本：

```text
【全局结局：解放者失去罗马】

元老院通过了对行动者的暂时大赦。

凯撒法令继续有效。

布鲁图斯在卡庇托利山宣布共和国已经恢复。

但城市没有跟随他。

凯撒的遗嘱被公开，平民与老兵看见自己仍在凯撒安排之中。

安东尼没有直接否定大赦。

他只让罗马人逐项回忆凯撒给过他们什么，又追问行动者准备拿走什么。

几日之内，密谋者从“解放者”变成需要离开罗马的人。

他们赢得了三月十五日。

却没有赢得第二天。
```

这是最接近历史回声的结局，但不要求完全复现历史后续。

```text
凯撒失去权力后的第一个下午，国库与公共文书被共同封存。

西塞罗主持紧急会议。

普通法令继续有效；
老兵土地不被立即改变；
终身独裁与无限任命权终止；
所有参与者接受元老院公开审查。

卡西乌斯也无法立即把胜利变成自己的权力。

共和国重新获得了运转能力。

但它必须在没有凯撒替所有人作决定的情况下，证明自己真的能够治理。
```

```text
却把凯撒留下的政治机器交给了最懂得使用它的人。

它只是换了一个尚未公开承认自己是主人的人。
```

# 6.9 布鲁图斯个人结局

## 大胜：共和国的制度奠基者

触发参考：

```text
全局结局 A、D 或高质量 F
共和声望 ≥ 75
信念裂痕 ≤ 45

西塞罗认可 ≥ 65
```

结局文本：

```text
【你的最终下场：共和国的制度奠基者】

你没有成为凯撒的继承人。

也没有成为卡西乌斯需要的神圣象征。

你迫使所有人承认：

结束个人统治并不等于恢复共和国；
恢复共和国必须先说明权力如何被限制、法令如何继续、军队如何服从。

你的名字没有只与三月十五日联系在一起。

它被写进了限制非常权力、保障公开复核与维持城市运转的协议。

你的主角类型：

制度型主角
危局仲裁者
拒绝两条既定道路的人
```

触发参考：

```text
罗马避免即时内战
```

结局文本：

```text
【你的最终下场：共和国的仲裁者】

你没有彻底击败任何一方。

凯撒、安东尼、卡西乌斯和西塞罗都认为你让自己失去了一部分原本可能得到的东西。

也正因为如此，没有一方能够独占结果。

你成为所有阵营都不完全信任，却又不得不继续接触的人。

你赢得的不是统治权。

是让罗马仍然存在下一次公开选择的时间。
```

## 悲剧胜利：解放者没有罗马

触发参考：

```text
凯撒死亡
玩家共和声望较高
行动获得部分正当性
但安东尼控制公共叙事或玩家被迫离城
```

结局文本：

```text
【你的最终下场：解放者没有罗马】

你完成了自己认为必须完成的行动。
```

# 9. AI 推演规则

## 9.1 AI 导演每次必须判断

```text
玩家真正想达成什么？
行动是公开还是私密？
玩家是否有身份、资源和接触权？
行动会留下什么证据或目击者？
谁因此受益？
谁因此受损？
谁会误解行动？
谁拥有反制能力？
它改变了哪一个世界变量？
是否生成新的 FateSeed？
是否推进、延迟或改写当前历史锚点？
是否影响第 7 幕合法性、现实控制或个人下场？
```

---

## 9.2 历史推演优先级

```text
第一优先：
人物利益与已知信息一致

第二优先：
玩家行动产生可追踪因果

第三优先：
世界状态达到阈值时触发反应

第四优先：
保留历史锚点作为默认压力

第五优先：
只有当玩家积累足够条件时才允许大幅改写历史
```

历史不能成为强制剧情：

```text
错误：
无论玩家做什么，凯撒必然死亡。

正确：
若玩家未改变警告、德西穆斯、安东尼、密谋凝聚和公开谈判条件，
历史路径拥有最高默认权重。
```

---

## 9.3 大幅改写历史的最低条件

### 凯撒接受限制

至少满足：

```text
临时共和国协议 ≥ 2/3
西塞罗认可 ≥ 60
共和声望 ≥ 65
凯撒信任 ≥ 30
内战风险 < 75
至少一个凯撒阵营角色愿意担保
```

### 密谋提前失败

至少满足：

```text
密谋暴露 ≥ 70
或警告信完整送达且安东尼警觉 ≥ 65
或玩家公开交出足够证据
```

### 行动后元老院成功控制罗马

至少满足：

```text
元老院合法性 ≥ 55
政治控制力 ≥ 55
城市秩序 ≥ 45
凯撒普通法令或老兵利益有明确保障
```

### 安东尼成功继承政治机器

至少满足：

```text
凯撒死亡
安东尼保持行动自由
安东尼警觉 ≥ 55
凯撒民望 ≥ 50
凯撒文书或广场至少控制其一
```

## 9.4 自定义行动 ActionGuard

允许：

```text
公开演说
私人谈判
调查文书与信息来源
使用元老院职位
联系家族与中立元老
设计有限政治协议
调整公开与私密信息
请求角色承担保证
利用已有证据施压
```

需要限制：

```text
直接命令不受玩家控制的军团
无条件拘留执政官或独裁官
凭空知道遗嘱内容
在无传播渠道时让全城立即知道消息
让角色违背核心利益却不给交换条件
用现代选举、媒体或宪法法院机制直接解决危机
```

拒绝示例：

```text
你的身份不足以直接命令所有罗马军团解除武装。

你可以改为：
联系已知军官；
向元老院提出命令；
用老兵利益保障交换中立；
要求凯撒或安东尼签署公开指令。
```

被拒绝的谋划不消耗谋划机会。

每幕必须至少出现：


不得出现：

```text
连续三条无决策价值的历史说明

凯撒只负责等待玩家决定
卡西乌斯无条件服从布鲁图斯
安东尼在凯撒死亡后不采取行动
西塞罗在没有制度方案时自动背书
```

## 13.3 因果验收

自动测试应检查：

```text
若玩家多次支持凯撒：
凯撒信任必须上升，卡西乌斯信任通常下降。

若玩家扩大密谋并延误警告：
密谋凝聚和行动成功权重上升，牵连与道德反噬也上升。

若玩家完成临时共和国协议：
第 6 幕必须允许真实妥协，而非强制历史结果。

若安东尼保持自由且控制文书：
第 7 幕必须显著提高安东尼继承或舆论反转权重。

若玩家废除全部凯撒法令：
军团与既得利益群体的敌意必须上升。

若玩家取消行动：
密谋不能自动消失，必须产生泄密、分裂或第二行动风险。

若密谋暴露极高：
凯撒不得毫无准备地进入历史路径。
```

# 14. 设计者备注

这个剧本真正玩的不是：

```text
凯撒会不会在三月十五日死亡。
```

而是：

```text
当一个最有能力、最受军队与群众支持的人
已经超越旧制度时，

反对者究竟应当：
等待制度自行恢复，
用不可逆行动终止他，
逼迫他接受限制，
还是与他共同创造一个不再是旧共和国的新秩序？
```

凯撒是公共危机。

布鲁图斯是道德与行动的入口。

卡西乌斯提供执行力，也制造行动失控的危险。

安东尼证明：

```text
一个人失去生命或权位，
不代表他建立的政治机器随之消失。
```

西塞罗证明：

```text
没有现实控制的合法性很脆弱，
没有合法性的现实控制也不会稳定。
```

德西穆斯证明：

```text
历史转折往往取决于一个同时被两边信任的人，
在最后时刻把哪一份承诺当真。
```

玩家最终必须感到：

```text
我不是在选择是否忠于凯撒。
我是在决定什么样的权力仍然可以被称为共和国。

我不是在复现历史。
我是在承担改变历史以后，第二天仍要治理罗马的责任。
```

使用原则：

```text

本剧本的决策、数值、隐藏目标、主动谋划和分支结局为产品原创设计。
不同古代作者对具体细节存在差异时，优先选择最有利于清晰因果与多人博弈的版本，并在剧情中保留“不确定信息”。
```

# 16. 最终产品定义

> **《凯撒：共和国最后的春天》是一场 1—6 人的历史命运推演。每名玩家都拥有不同的共和国、不同的私人利益和不同的危险。凯撒的命运会被改变，但真正的胜负发生在第二天：谁能让罗马继续运转，谁才能决定历史如何记住三月十五日。**
