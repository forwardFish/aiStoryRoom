# Many Worlds 图标裁剪包

共 46 个图标，已从原始图标总览图中逐个提取并清理。

## 文件夹

- `png-tight/`
  - 按图标实际边界裁剪
  - 保留少量透明边距
  - 不做不必要的放大，细节最接近原图
- `png-128/`
  - 128×128 透明 PNG
  - 推荐用于网页导航、按钮和功能模块
- `png-256/`
  - 256×256 透明 PNG
  - 推荐用于高分屏、较大卡片和后续缩放
- `preview.png`
  - 46 个图标的质检预览
- `manifest.json`
  - 图标名称、顺序、源图坐标和输出路径

## 处理结果

- 已删除卡片圆角边框
- 已删除英文标签文字
- 已删除浅色背景
- 所有 PNG 均为真实透明背景
- 图标主色统一保留为原图紫色 `#7B51FF`
- 文件名已按页面顺序编号，便于 Codex 自动映射
